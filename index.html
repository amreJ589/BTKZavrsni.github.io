Uvod u SVG

SVG (Scalable Vector Graphics) je XML baziran format za dvodimenzionalnu vektorsku grafiku na webu.

Za razliku od raster formata (poput JPG-a ili PNG-a) koji su građeni od mreže piksela, SVG je građen od matematičkih formula. To znači da preglednik doslovno crta elemente na ekranu na temelju zadanih koordinata.

  • Beskonačna skalabilnost: Ne gubi kvalitetu pri zumiranju, uvijek je 100% oštar bez obzira na veličinu ekrana.
  • Mala veličina datoteke: Često zauzima puno manje prostora od klasičnih slika jer je riječ o običnom tekstu.
  • Programabilnost: Elementi se mogu lako animirati pomoću CSS-a i JavaScripta jer su dio DOM stabla.
  • Dostupnost: Tekst unutar SVG-a je čitljiv alatima za pristupačnost i pretraživačima (SEO).
<!-- Jednostavan SVG krug --> <svg width="100" height="100"> <circle cx="50" cy="50" r="40" stroke="#10b981" stroke-width="4" fill="#34d399" /> </svg>

Geometrijski oblici

U SVG-u možete direktno crtati osnovne geometrijske oblike koristeći ugrađene tagove. Svaki oblik zahtijeva specifične atribute za pozicioniranje i dimenzije.

  • <rect> - Pravokutnik. Koristi x i y za početnu točku, te width i height. Atributi rx i ry zaobljuju rubove.
  • <circle> - Krug. Definira se centrom (cx, cy) i radijusom (r).
  • <ellipse> - Elipsa. Slična krugu, ali ima horizontalni (rx) i vertikalni radijus (ry).
  • <line> - Linija koja povezuje početnu točku (x1, y1) sa završnom točkom (x2, y2).
  • <polygon> - Oblik s više ravnih linija zadanih kroz atribut points.
<!-- Više oblika u jednom SVG-u --> <svg width="150" height="100"> <rect x="10" y="25" width="50" height="50" rx="10" fill="#3b82f6" /> <ellipse cx="110" cy="50" rx="30" ry="20" fill="#8b5cf6" /> </svg>

Moćna Putanja (Path)

Element <path> je najsvestraniji i najmoćniji element u SVG-u. S njim možete nacrtati bilo kakav kompleksni, nepravilni oblik, poput ikona, slova ili našeg SVG psa iz glavnog izbornika!

Sva čarolija nalazi se u atributu d (data), unutar kojeg koristimo posebne naredbe (slova) i koordinate. Najvažnije naredbe:

  • M (Move To): Pomakni "olovku" na početnu koordinatu bez crtanja (npr. M 10 10).
  • L (Line To): Povuci ravnu liniju od trenutne do nove koordinate (npr. L 50 10).
  • C ili Q (Curve): Crtanje matematičkih krivulja za glatke, zakrivljene linije.
  • Z (Close Path): Zatvori putanju povlačenjem ravne linije nazad do prve točke.
<!-- Crtanje jednostavnog trokuta putanjom --> <svg width="100" height="100"> <path d="M 50 10 L 90 90 L 10 90 Z" fill="none" stroke="#f59e0b" stroke-width="5" /> </svg>

Stiliziranje (Boje i Linije)

Stiliziranje SVG-a vrlo je slično CSS-u, ali se koriste specifični atributi (ili CSS svojstva) namijenjena vektorskoj grafici.

  • fill: Određuje boju kojom se ispunjava unutarnji prostor oblika (može biti boja, HEX kod ili transparent / none).
  • stroke: Određuje boju linije (okvira) oko oblika.
  • stroke-width: Određuje debljinu vanjske linije.
  • stroke-dasharray: Razbija punu liniju u crtice i praznine (pogledaj primjer). Ovo je ključno za animacije "iscrtavanja" ikona.
  • stroke-linecap: Određuje kako izgleda završetak linije (npr. round za glatke, zaobljene krajeve).
<!-- Primjer isprekidane deblje linije --> <svg width="100" height="100"> <path d="M 20 50 L 80 50" stroke="#ef4444" stroke-width="15" stroke-linecap="round" stroke-dasharray="1 30" /> </svg>

CSS Animacije nad SVG-om

Ono što SVG čini apsolutno superiornim formatom za moderan web je njegova potpuna kompatibilnost s CSS-om i JavaScriptom. Možete primijeniti standardni CSS @keyframes sustav na bilo koji unutarnji SVG element.

Najčešće animacije uključuju promjenu boje (fill), skaliranje, rotaciju, pomicanje po osima ili spomenuto "iscrtavanje" putanja.

/* CSS klasa dodijeljena SVG krugu */ .pulsiraj-krug { transform-origin: center; animation: puls 1.5s infinite alternate; } @keyframes puls { from { transform: scale(0.8); opacity: 0.5; } to { transform: scale(1.2); opacity: 1; } }
